Project Name: Useful Scripts



A collection of personal automation scripts for various productivity tasks.



Current Feature: Rapid Desktop Cleaner (AutoHotkey v2)



cleandesk.ahk: The core script. It utilizes Windows APIs for high-performance toggling. Use Alt + Z to instantly hide or show your desktop icons and the taskbar.



cleandeskauto.ahk: The automation version. This script includes "Run at Startup" support and a SetTimer routine that checks every 500ms to ensure the desktop and taskbar remain hidden, providing a consistently clean workspace.
